¿Cómo registrar un producto?

  1. Haz clic en "Registrar Producto".
  2. Llena el formulario con los datos del producto.
  3. ¡Listo! El producto aparecerá en la lista.

Requisitos del Código de Producto

El código debe cumplir con estos formatos válidos:

  • 3 letras + 4 números (ej: MED1234)
  • Primeras 3 letras deben ser mayúsculas
  • No se permiten caracteres especiales

Ejemplos aceptados:

ABC1001 MED2023 VIT3045
